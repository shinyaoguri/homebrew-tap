# Shinyaoguri Tap

## How do I install these formulae?

`brew install shinyaoguri/tap/<formula>`

Or `brew tap shinyaoguri/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "shinyaoguri/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
